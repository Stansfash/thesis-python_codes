# Chapter 5: Sentinel-2 Based Biomass Prediction with Interpolation

Remote sensing analysis using Sentinel-2 satellite imagery for pasture biomass prediction.

## Files
- `Chapter_5_data_cleaning.ipynb` - Data preprocessing and merging
- `Chapter_5_modelling_visualisations.ipynb` - Model training and analysis

## Notebook 1: Data Cleaning
- Merge plate meter, Sentinel-2, weather, and grazing data
- Temporal alignment
- Multiple interpolation strategies
- Validation set creation

## Notebook 2: Modelling
- Feature selection experiments
- Interpolation method comparison
- Temporal transferability assessment
- Progressive training
- Farm-level performance analysis

## Usage
```bash
jupyter notebook Chapter_5_data_cleaning.ipynb
jupyter notebook Chapter_5_modelling_visualisations.ipynb
```
